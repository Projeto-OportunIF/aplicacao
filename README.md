### **Resumo do Projeto**
Atualmente, os discentes do IFPR precisam cumprir atividades complementares ou estágios obrigatórios para sua formação acadêmica. Contudo, a divulgação dessas oportunidades pelos docentes ocorre de forma dispersa, por meios como WhatsApp, e-mail ou avisos em sala de aula. Esses canais nem sempre alcançam todos os alunos, além de dificultarem a organização e o gerenciamento das inscrições. Diante desse cenário, surgiu o OportunIF, uma solução digital voltada a centralizar e otimizar a divulgação de oportunidades acadêmicas, visando facilitar  tanto o acesso dos estudantes quanto o controle por parte dos professores. O sistema tem como objetivo centralizar todas as oportunidades e permite que os estudantes realizem suas inscrições de forma prática e segura, acompanhando o status. Para os professores, o sistema oferece recursos de cadastro detalhado das oportunidades, com informações claras sobre requisitos, prazos, documentos necessários e cursos elegíveis. Para o desenvolvimento seguimos a abordagem de construção de um sistema web, utilizando arquitetura MVC (Model-View-Controller), com as ferramentas como JavaScript, PHP, HTML, CSS, e MySQL para o gerenciamento do banco de dados e além das técnicas de análise de requisitos para levantar as necessidades de alunos e professores, possibilitando entender e atender as necessidades dos discentes e docentes. O desenvolvimento do OportunIF resultou em um sistema capaz de centralizar a divulgação e o gerenciamento das oportunidades acadêmicas no IFPR.
